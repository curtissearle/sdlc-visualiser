## SDLC Visualiser

A lightweight web app for quickly creating, visualising, and sharing SDLC maps (stages, gates, artifacts, and tooling) with a focus on speed, clarity.

## Prerequisites

- Node.js 18+ and pnpm (or npm/yarn)

## Getting Started

### 1. Install

```bash
pnpm install
```

### 5. Run the Development Server

```bash
pnpm run dev
```
