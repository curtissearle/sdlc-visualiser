## Prerequisites

- Node.js 18+ and pnpm (or npm/yarn)

## Getting Started

### 1. Install

```bash
pnpm install
```

### 2. Run the Development Server

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.
